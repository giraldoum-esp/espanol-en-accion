Subir material nuevo

  1. Ve a tu repositorio en GitHub: Code → Add file → Upload files.
  2. Arrastra tu archivo .html, .pdf o .svg y pulsa Commit changes.
  3. Para que aparezca en el Buscador, edita recursos.html y añade una línea como:
    ["Fichas — Condicionales","ficha_condicionales.html","gramática B1 condicionales"],
  4. Para mostrarlo en la portada, edita index_v6.html y copia/pega un bloque de tarjeta.

Consejo: usa nombres claros, por ejemplo: ficha_condicionales.html, lectura_b1_viajes.html.
